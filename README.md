# Ontology Ink Adventure

Dynamic Interactive Fiction using Ollama + Ontology + Ink + inkjs

Built on top of Stanestane/Ontology_Character_AI

## Features
- Sassy bratty narrator Saya
- Ontology-driven consistency
- Incremental Ink story generation
- Browser-based player with inkjs

## Setup
1. Clone this repo
2. `pip install fastapi uvicorn ollama`
3. Run `ollama serve` and pull a model (e.g. llama3.2)
4. `uvicorn app:app --reload`
